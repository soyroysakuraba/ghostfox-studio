# Como publicar esta web para Google Play

Esta carpeta contiene la web publica de Ghostfox Studio / INKARA.

Archivos principales:

- `index.html`
- `privacidad.html`
- `eliminacion-datos.html`
- `reglas-comunidad.html`
- `soporte.html`
- `styles.css`

## URL que debes usar en Google Play

Cuando la web este publicada, usa la URL de:

`privacidad.html`

Ejemplo:

`https://tuusuario.github.io/ghostfox-studio/privacidad.html`

## Opcion recomendada: GitHub Pages

1. Crea un repositorio publico llamado `ghostfox-studio` o `inkara`.
2. Sube todos los archivos de esta carpeta.
3. En GitHub, entra a Settings > Pages.
4. En Source selecciona `Deploy from a branch`.
5. Selecciona la rama `main` y carpeta `/root`.
6. Guarda y espera a que GitHub genere la URL.
7. Copia la URL de `privacidad.html` en Play Console.

## Otras opciones

Tambien puedes usar Google Sites, Netlify, Vercel o cualquier hosting estatico.

Lo importante para Google Play:

- La pagina debe ser publica.
- No debe requerir login.
- Debe abrir desde navegador movil y escritorio.
- Debe incluir contacto de soporte.
- Debe mantenerse disponible mientras INKARA este publicada.
